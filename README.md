# 🎓 AI-Based Career & Lifestyle Planner

An interactive web application designed to generate personalized career and lifestyle plans for students from all academic backgrounds.

This system provides customized recommendations based on the user's:

- Course of Study
- Stream of Study
- Year of Study
- Area of Interest
- Budget Range of Students
- Diet Preferences

It also includes a structured feedback system with 5-star ratings to evaluate user satisfaction.

---

## 🚀 Features

### ✅ Personalized Career Planning
- Course-specific and stream-based guidance
- Year-wise academic roadmap
- Budget-based learning recommendations
- Career enhancement strategies

### 🍎 Lifestyle & Diet Planning
- Separate meal suggestions:
  - Morning
  - Afternoon
  - Evening
  - Night
- Healthcare tips for better well-being

### ⭐ 10-Question Feedback System
- 5-star rating system (1 to 5 scale)
- Automatic average rating calculation
- Dynamic response based on feedback:
  - Satisfactory (4–5 stars)
  - Moderate (3 stars)
  - Needs Improvement (1–2 stars)

### 🎨 User Interface
- Multi-page interactive flow
- Clean professional design
- Smooth navigation between sections

---

## 🛠 Technologies Used

- **Programming Language:** Python
- **Framework:** Streamlit
- **UI Styling:** HTML & CSS (custom styling)
- **Deployment Platform:** Streamlit Community Cloud
- **Version Control:** GitHub
